void canvas_miterLimit ( VGfloat  miter_limit)

The miterLimit property of the Canvas 2D API sets the miter limit ratio in space units. Setting zero, negative, Infinity and NaN values are ignored; otherwise the current value is set to the new value.

Parameters
miter_limitA number specifying the miter limit ratio in space units. Zero, negative, Infinity and NaN values are ignored.
